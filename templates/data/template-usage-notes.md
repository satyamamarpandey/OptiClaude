# Template Usage Notes

## Fast pattern
1. Pick the closest template.
2. Replace the bracketed placeholders.
3. Paste it into Claude Code.
4. Keep the scope narrow.
5. Ask for verification at the end.

## Good habit
Pair a task template with one data file:

- `feature-request.md` + `task-brief.example.json`
- `code-review.md` + `verification-checklist.example.json`
- `project-kickoff.md` + `repo-profile.example.json`

## Why this helps
This reduces ambiguity, improves consistency, and makes Claude less likely to over-read the codebase.
